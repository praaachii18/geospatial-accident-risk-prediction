
# Geospatial Road Accident Risk Analysis & Prediction

## Overview
This project analyzes road accident data and predicts accident severity using machine learning and geospatial visualization.

## Features
- Data preprocessing and analysis using Pandas
- Interactive maps using Folium
- Accident hotspot detection
- Random Forest model for severity prediction
- Visualization of trends (weather, time, road type)

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Folium, GeoPy
- Matplotlib

## Output
- Accident hotspot maps
- Severity prediction model
- Data insights for road safety

## How to Run
1. Open the notebook in Google Colab
2. Install required libraries
3. Run all cells
